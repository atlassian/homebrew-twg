Installation

Warning

Early Access Program (EAP): TWG CLI is not generally available. To request access, contact your team lead or submit a request via the EAP signup form.

Before you begin

  • You need macOS (arm64 or x64), Linux (x64), or Windows (arm64 or x64).
  • You need an Atlassian account with access to the products you want to use.
  • You need an AI coding agent installed on your computer. This guide uses Claude Code as an example. For other supported agents, see What is TWG CLI?.
  • You need Early Access Program (EAP) access to TWG CLI — see the notice at the top of this page.

The TWG CLI binary does not require a separate runtime (for example, no Node.js).

Install an AI coding agent

TWG CLI works with any AI coding agent. This guide uses Claude Code as an example.

If you already have an AI coding agent installed, skip this section and continue with macOS and Linux or Windows.

Install Claude Code

Note

The Homebrew and shell script options below are for macOS and Linux. On Windows, follow the install steps for Claude Code (or your chosen agent) from the vendor's documentation.

  1. Open Terminal.

  2. Run:

    brew install claude-code

    This requires Homebrew. If you do not have Homebrew, you can install Claude Code with:

    curl -fsSL https://claude.ai/install.sh | bash

  3. Follow the prompts to complete the installation.

macOS and Linux

Run the install script:

bash <(curl -fsSL https://teamwork-graph.atlassian.com/cli/install)

The installer detects your platform, verifies the download checksum, and places the binary in ~/.local/bin.
It then runs twg login to authenticate. It also runs twg skills install to install the twg skill for your agent.
If ~/.local/bin is not in your PATH, the installer prints the exact line to add to your shell profile.

Note

When prompted for credentials, see Authentication for instructions on creating an API token.

Homebrew (macOS)

brew tap atlassian/twg
brew install twg
twg login
twg skills install

Windows

Note

EAP note: Windows may show a SmartScreen warning — click More info → Run anyway. This is expected for EAP. GA binaries will be signed.

Install — PowerShell

iwr https://teamwork-graph.atlassian.com/cli/setup.ps1 -UseBasicParsing | iex

Install — Command Prompt (cmd)

Note

curl is built into Windows 10 1803 and later. If you're on an older version, use the PowerShell method above instead.

powershell -ExecutionPolicy Bypass -Command "iwr https://teamwork-graph.atlassian.com/cli/setup.ps1 -UseBasicParsing | iex"

Login and setup

Once installed (either shell), run:

twg login
twg skills install

Verify the installation

twg doctor

This works the same on macOS, Linux, PowerShell, and cmd.

Update

macOS/Linux — use the built-in updater:

twg update

macOS (Homebrew):

brew update && brew upgrade twg

Windows (PowerShell or cmd) — re-run the download command from the install section above, replacing the existing .exe.

After any upgrade, re-run skill installation to pick up updated skill files:

twg skills install
